# 🎯 Smart Interview Preparation System

An interactive# 🎯 Smart Interview Preparation System

🚀 A modern web application that helps users prepare for technical interviews with personalized study plans, role-based guidance, and interactive insights.

---

## ✨ Features

✔ Role-based preparation (Software Engineer, Data Analyst, etc.)  
✔ Skill-level recommendations (Beginner / Intermediate / Advanced)  
✔ Dynamic topic suggestions for each role  
✔ Curated interview questions for practice  
✔ Clean, responsive, and interactive UI  
✔ Focus area identification for improvement  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Pandas  
- Matplotlib
